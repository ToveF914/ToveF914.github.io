# Flex 弹性布局

flex-direction 决定主轴的方向
flex-wrap 决定如果一条轴线排不下，如何换行
justify-content 定义了项目在主轴上的对齐方式
align-items 属性定义项目在交叉轴上如何对齐
align-content属性定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用
